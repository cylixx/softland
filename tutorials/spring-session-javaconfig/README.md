Spring Session + Redis + Servlet Integration Example
===============================
his page will walk through Spring session, Redis and Servlet integration example.


###1. Technologies used

* spring-boot-starter 1.4.3.RELEASE
* lettuce 3.5.0.Final
* Maven 3

###2. To Run this project locally
```shell
$ mvn clean install -e
$ mvn tomcat7:deploy
```
```shell
* Test page 
   http://localhost:9080/spring-session/
```

###3. To import this project into Eclipse IDE
1. ```$ mvn eclipse:eclipse```
2. Import into Eclipse via **Existing Maven Projects** option.
3. Done.
