Spring + Redis basic annotation
===============================
Introduction to Spring Data Redis example

###1. Technologies used
* spring-boot 1.2.0.RELEASE
* Maven 3

###2. To Run this project locally
```shell
mvn clean install -e
mvn test
```

###3. To import this project into Eclipse IDE
1. ```$ mvn eclipse:eclipse```
2. Import into Eclipse via **Existing Maven Projects** option.
3. Done.
