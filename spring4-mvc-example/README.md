Spring 4 MVC crud Example
===============================
Template for Spring 4 MVC crud.

###1. Technologies used
* Spring 4.2.2.RELEASE
* Jackson 2.6.3
* Maven 3

###2. To Run this project locally
```shell
$ mvn jetty:run
```
listAllUsers ```http://localhost:8080/spring4-mvc-example/user/```
createUser ```http://localhost:8080/spring4-mvc-example/user/```


###3. To import this project into Eclipse IDE
1. ```$ mvn eclipse:eclipse```
2. Import into Eclipse via **Existing Maven Projects** option.
3. Done.
