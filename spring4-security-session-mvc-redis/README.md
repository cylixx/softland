
# Spring Security Annotation y su configuración
La necesidad de usar Spring Security Annotation y eliminar los XML esta creciendo . La mayoría de nosotros usamos Spring 4.X a la espera del salto que depare la versión 5 con flux etc. Sin embargo a veces es difícil encontrar documentación clara de como configurar el framework a nivel de seguridad utilizando anotaciones y eliminando los XML.

*   [Spring Security Annotation y su configuración](https://www.arquitecturajava.com/spring-security-annotation-y-su-configuracion/ "title")
*	[Datos de sesión externalizados con Spring Session](https://picodotdev.github.io/blog-bitix/2015/03/datos-de-sesion-externalizados-con-spring-session/ "title")  
*	[Getting started with Spring Security, Spring Session and Redis](https://drissamri.be/blog/2015/05/21/spring-security-and-spring-session/ "title")  



##### To Run this project locally
```shell
 curl -v http://localhost:9080/spring4-security-session-mvc-redis/ -u user:password
 curl -v http://localhost:9080/spring4-security-session-mvc-redis/hola -H "x-auth-token: 8475b6a9-b106-4026-9752-da4e21d8e641"
```
